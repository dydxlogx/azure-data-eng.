# 24-Week Data Engineer Learning Path (Product Company Standard)

## Phase 1 (Weeks 1-6): Foundations
- Advanced SQL (window functions, tuning, indexing).
- Data modeling basics (OLTP vs OLAP, star schema).
- Python + PySpark essentials.
- Version control and CI basics.

## Phase 2 (Weeks 7-12): Platform Skills
- ADF orchestration and parameterization.
- Databricks Delta fundamentals and optimization.
- Azure SQL performance and security.
- Power BI modeling and DAX.

## Phase 3 (Weeks 13-18): Production Engineering
- Observability, logging, alerting, incident response.
- Data quality engineering frameworks.
- Governance and cataloging with Purview.
- IaC and environment promotion.

## Phase 4 (Weeks 19-24): Senior-Level Practices
- Cost optimization (cluster right sizing, storage lifecycle).
- Platform architecture design reviews.
- Data contracts and product metrics.
- Mentoring and writing design docs.

## Weekly Routine
1. Build one small feature.
2. Add tests and observability.
3. Write a mini design doc and trade-offs.
4. Run post-mortem practice for one failure scenario.
